Non-Billable Snapshot

Internal Product Tracks and Technical Document Footprints

The non-billable archive spans three products during Internship Capstone-II and two products in CRW-I. The source folders include Technical Documents, FDDs, SuiteScript packages, Object metadata, and batch processing logic used to shape approvals, segmentation rules, billing validations, and email/process automation.

Internship Capstone-II · Product-1

Subscription and invoice creation utilities backed by grouped JSON structures, helper libraries, and SuiteScript processing.

Internship Capstone-II · Product-2

Validation-before-billing and validation-before-invoicing flows built with map/reduce style processing and request-context data files.

Internship Capstone-II · Product-3

Bulk emailing logic with SuiteScripts and Technical Document support for batched communications.

CRW-I · Product-1

Purchase Price Review functionality documented with an FDD and implemented using user event, map/reduce, and saved search automation.

CRW-I · Product-2

Mandatory segmentation logic with technical and functional documents plus SuiteScript enforcement across GL account conditions.
